# 🚀 LKSN Cloud Computing Project - Laila Cloud

Proyek ini adalah implementasi infrastruktur cloud AWS untuk aplikasi web Guestbook berbasis Python Flask.

## 🏗️ Architecture Overview
- **Compute:** EC2 Instance (Private Subnet) with Application Load Balancer (Public Subnet).
- **Storage:** Amazon S3 for static file uploads.
- **Database:** Amazon RDS (MySQL) for guestbook data storage.
- **Serverless:** AWS Lambda & API Gateway integration.
- **Security:** IAM Roles for S3 access & Security Groups for networking.

## 🛠️ Tech Stack
- **Backend:** Python (Flask, Boto3, PyMySQL)
- **Frontend:** HTML5, CSS3 (Modern AWS-inspired UI)
- **Database:** MySQL (AWS RDS)
- **DevOps:** Git, SSH, AWS CLI
